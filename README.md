●アプリ概要

本アプリは、Javaで作成したスケジュール管理のコンソールアプリです。 ユーザーごとに予定を管理できるようにしました。 また、ログイン機能やデータ永続化を実装しています。

●作成目的

・avaの基本文法の実践だけでなく以下のことを意識して開発しました。 ・スケジュール表の基本動作予定の追加・検索・削除の実装 ・ログイン機能を実装することでユーザー単位のデータ管理 ・メニュー／サブメニューを実装して選択画面での画面を見やすくする ・ファイルによるデータの永続化 ・エラー入力に対する耐性

●主な機能

・ログイン機能 　 ユーザーを選択してログイン ログアウト後は再ログイン可能

●予定管理 　 ・予定の追加 　・予定の検索 　・予定の削除 　・自分の予定一覧表示

●ユーザー管理 　 ・ユーザーの追加 　・ユーザーの削除 　・ユーザーの重複防止 　・ユーザー名の並び替え

●データ保存

users.txtを作成しユーザー情報を保存 schedule.textを作成し予定情報を保存

●パスワード設定（平文）

・ユーザー作成時に簡易的なパスワードを追加

●工夫した点

１：サブメニュー追加 　１つのメニューに全ての操作を一覧にした時に選択肢で画面が圧迫されていました。 そのためメニューとサブメニューに分けることで操作の見通しを良くしました。

２：ログイン状態の管理 　 loginUserを用いてログインユーザーのみで操作できるようにしました。 メニューにログアウト操作ができるのでユーザー切り替えも可能です。

３：日本語の並び替順対応 　Collator(Locale.JAPANESE)を使用し、ユーザー名を五十音順で表示しています。

４：データ保存のタイミング 　ユーザー追加・削除・ログアウト・終了時にファイルの保存を行い、データの整合性を保っています。
